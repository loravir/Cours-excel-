0 C'est quoi un tableur ? Pourquoi Excel ?

🖥️ Définition simple

Un tableur est un logiciel qui permet de faire des calculs automatiquement dans un tableau. On y écrit des données (chiffres, textes) dans des cases, et on lui dit quelles opérations effectuer.

Imagine : Tu as un grand cahier quadrillé. Chaque petite case peut contenir un nombre ou un mot. Et si tu écris une formule magique dans une case, elle calcule toute seule le résultat — même si tu changes les données !

Les tableurs les plus utilisés sont :

Microsoft Excel (le plus célèbre) OpenCalc (Open Office) Sun StarOffice Calc KSpread (Linux) Lotus 1-2-3

Dans ce cours, on travaille avec Microsoft Excel.

I L'interface d'Excel — Connaître son environnement

Les éléments clés à connaître

A B C D E
1 ← Cellule A1 B1 C1 D1 E1
2 A2 ← Cellule B2 C2 D2 E2
3 A3 B3 ← C3 active D3 E3

Ce que tu vois dans Excel :

  • COLONNES Les colonnes sont les lignes verticales, nommées avec des lettres : A, B, C, D, E… Z, AA, AB…
  • LIGNES Les lignes sont les rangées horizontales, numérotées : 1, 2, 3, 4…
  • CELLULE Une cellule = intersection d'une colonne et d'une ligne. Exemples : A1, B3, C15. C'est la case où tu saisis tes données.
  • PLAGE Une plage de cellules = un groupe de cellules. Exemple : A1:E5 désigne toutes les cellules de A1 à E5.
  • BARRE DE FORMULE C'est la zone en haut où tu vois et tapes les formules. Si tu es dans la cellule C3 et que tu tapes une formule, elle s'affiche ici.
Analogie : Une cellule Excel, c'est comme une case d'un carnet quadrillé. Son nom (A1, B3…) c'est son adresse — comme le numéro d'une maison dans une rue.
II Les Références de Cellules

🔗 Deux types de références à connaître

Quand on écrit une formule, on utilise les adresses (références) des cellules. Il en existe deux types :

📍 Référence RELATIVE

Elle change automatiquement quand tu copies-colles la formule dans une autre cellule.

=A1+B1
Si tu copies cette formule en ligne 2 → elle devient =A2+B2
🔒 Référence ABSOLUE

Elle ne change JAMAIS, même si tu copies la formule. On utilise le signe $.

=$A$1+B1
Peu importe où tu copies, $A$1 reste fixe !
Explication simple : La référence relative, c'est comme dire "prends la case juste à gauche". Elle suit le mouvement. La référence absolue, c'est comme dire "prends TOUJOURS la case A1, sans exception". Elle est fixée.
La notation L1C5 (Ligne 1, Colonne 5) est aussi valide dans Excel. Elle pointe vers la cellule à l'intersection de la ligne 1 et de la colonne 5 (= cellule E1).
III Les Fonctions de Base

📐 Qu'est-ce qu'une fonction ?

Une fonction est une formule prédéfinie qui effectue un calcul précis. Au lieu d'écrire =A1+A2+A3+A4+A5, tu peux écrire =SOMME(A1:A5) — c'est plus rapide !

La syntaxe générale est : =NOM_FONCTION(argument1; argument2; …)

➕ SOMME() — Additionner des cellules

Elle additionne toutes les valeurs d'une plage de cellules.

Syntaxe
=SOMME(cellule_début : cellule_fin)
Additionne toutes les cellules entre début et fin
Exemples
=SOMME(A1:A10)
Additionne A1 + A2 + A3 + … + A10
=SOMME(A1;A3;A5:A9)
Additionne A1, A3, puis toutes les cellules de A5 à A9
Image : C'est comme faire la somme de tous tes points dans un jeu vidéo, sans les compter un par un !

📊 MOYENNE() — Calculer la moyenne

Elle calcule la moyenne arithmétique d'une plage. Elle additionne tout et divise par le nombre de valeurs.

Syntaxe
=MOYENNE(A1:A10)
= (A1+A2+…+A10) ÷ 10
Image : C'est la moyenne de classe ! Tu additionnes toutes les notes et tu divises par le nombre d'élèves.

🔀 SI() — Tester une condition

C'est la fonction la plus puissante pour les décisions. Elle dit : "Si tel truc est vrai, fais ça, sinon fais autre chose."

Syntaxe
=SI(condition ; valeur_si_vrai ; valeur_si_faux)
Exemple concret
=SI(B2>=10 ; "Admis" ; "Refusé")
Si la note en B2 est ≥ 10 → affiche "Admis", sinon affiche "Refusé"
Image : C'est comme un agent de contrôle : "Si ton ticket est valide → tu passes. Sinon → tu ne passes pas."

➕🔀 SOMME.SI() — Additionner sous condition

Elle additionne uniquement les cellules qui respectent une condition.

Syntaxe
=SOMME.SI(plage_test ; critère ; plage_somme)
Exemple
=SOMME.SI(B2:B10 ; "Toyota" ; C2:C10)
Additionne les ventes (colonne C) uniquement pour les lignes où la marque (colonne B) est "Toyota"

🏆 RANG() — Classer des valeurs

Elle donne le rang (classement) d'une valeur dans une liste.

Syntaxe
=RANG(valeur ; référence ; ordre)
ordre = 0 → décroissant (le plus grand = rang 1) | ordre = 1 → croissant
Exemple
=RANG(B2 ; $B$2:$B$10 ; 0)
Donne le rang de B2 parmi toutes les notes de B2 à B10 (en ordre décroissant)
Remarque : on utilise $B$2:$B$10 (référence absolue) pour que la plage ne change pas quand on copie la formule vers le bas.
IV Mise en Forme & Opérations Avancées

🎨 La Mise en Forme d'un Tableau

La mise en forme, c'est embellir ton tableau pour qu'il soit lisible et professionnel. Excel propose des styles de tableau prédéfinis.

  1. Sélectionne les cellules que tu veux mettre en forme
  2. Va dans l'onglet Accueil → clique sur Mettre sous forme de tableau
  3. Choisis un style dans la liste (couleurs, rayures…)

Tu peux aussi personnaliser manuellement : gras, couleur de fond, bordures, taille de police, alignement…

🔒 Protéger une Feuille de Calcul

Protéger une feuille permet d'empêcher des personnes non autorisées de modifier tes données.

  1. Sélectionne les cellules qui peuvent être modifiées
  2. Dans l'onglet Accueil → Cellule → Format → désactive "Verrouiller la cellule"
  3. Clique sur Protéger la feuille
  4. Tape un mot de passe et confirme-le
Image : C'est comme mettre un cadenas sur ton classeur : seules les personnes avec la clé (le mot de passe) peuvent modifier le contenu.

🖨️ Les Zones d'Impression

La zone d'impression définit exactement quelles parties du tableau seront imprimées sur papier.

  1. Sélectionne les cellules à imprimer
  2. Va dans l'onglet Mise en Page
  3. Clique sur Zone d'impression → Définir
Pratique pour n'imprimer qu'une partie du tableau, pas toute la feuille vide !
V 🧪 Exercice Pratique Complet

Contexte de l'exercice

Le Lycée Général de Douala organise la remise de notes du premier trimestre. Le professeur principal te demande de créer un tableau Excel pour gérer les résultats de 8 élèves dans 4 matières.

Voici le tableau tel qu'il devrait apparaître dans Excel :

A B C D E F G H I
1 N° Nom de l'élève Maths Physique Français SVT Moyenne Mention Rang
2 1 MBALLA Eric 15 13 12 14 =MOYENNE(C2:F2) =SI(G2>=10;"Admis";"Refusé") =RANG(G2;$G$2:$G$9;0)
3 2 FOUDA Marie 18 17 15 16 16.5 Admis 1
4 3 NDJOCK Paul 8 7 11 9 8.75 Refusé 8
5 4 BELINGA Sophie 14 15 13 12 13.5 Admis 3
6 5 ATEBA Jean 11 10 9 12 10.5 Admis 6
7 6 OWONA Cécile 16 14 17 15 15.5 Admis 2
8 7 MANGA Boris 12 11 10 13 11.5 Admis 5
9 8 ESSONO Diane 13 12 14 11 12.5 Admis 4
10 MOYENNE CLASSE =MOYENNE(C2:C9) =MOYENNE(D2:D9) =MOYENNE(E2:E9) =MOYENNE(F2:F9) 12.84
11 TOTAL MATHS CLASSE =SOMME(C2:C9)
Les cellules en jaune/italique contiennent des formules. Les cellules en vert clair montrent les résultats calculés.
VI 📝 Questions sur l'Exercice
1
Quelle est la référence de la cellule contenant la note de Maths de FOUDA Marie ?

Observe le tableau et repère la ligne de FOUDA Marie et la colonne Maths.

FOUDA Marie est en ligne 3. La colonne Maths est la colonne C.

Donc la référence est : C3

La valeur dans cette cellule est 18.

2
Écris la formule Excel pour calculer la moyenne de MBALLA Eric (ligne 2).

Sa note de Maths est en C2, Physique en D2, Français en E2, SVT en F2.

La formule à taper dans la cellule G2 est :

=MOYENNE(C2:F2)

Cette formule additionne C2, D2, E2, F2 puis divise par 4. Résultat : (15+13+12+14) ÷ 4 = 13.5

On aurait aussi pu écrire : =(C2+D2+E2+F2)/4 — c'est équivalent, mais moins élégant.

3
Écris la formule pour afficher "Admis" si la moyenne est ≥ 10, et "Refusé" sinon (pour l'élève en ligne 2).

La formule à taper dans la cellule H2 est :

=SI(G2>=10 ; "Admis" ; "Refusé")

Explication :

• G2>=10 → c'est la condition testée

• "Admis" → ce qui s'affiche si la condition est vraie

• "Refusé" → ce qui s'affiche si la condition est fausse

N'oublie pas les guillemets autour des mots (textes) dans la formule SI !
4
Écris la formule pour donner le rang de MBALLA Eric (G2) dans la liste des moyennes G2:G9. On veut que le meilleur soit 1er.

La formule à taper dans la cellule I2 est :

=RANG(G2 ; $G$2:$G$9 ; 0)

Explication :

• G2 → la valeur dont on cherche le rang

• $G$2:$G$9 → la plage de toutes les moyennes (en référence absolue avec $)

• 0 → ordre décroissant : la plus grande moyenne = rang 1

On DOIT mettre $G$2:$G$9 avec les $ (référence absolue). Sans ça, quand tu copies la formule vers le bas, la plage se décale et les rangs seront faux !
5
Quelle formule calcule la moyenne de toute la classe en Maths (colonne C, lignes 2 à 9) ?

La formule à taper dans la cellule C10 est :

=MOYENNE(C2:C9)

Elle calcule la moyenne de toutes les notes de Maths : (15+18+8+14+11+16+12+13) ÷ 8 = 13.38

6
Comment afficher une mention plus précise ? Écris une formule SI imbriquée : "Très Bien" si moy ≥ 16, "Bien" si moy ≥ 14, "Assez Bien" si moy ≥ 12, "Passable" si moy ≥ 10, sinon "Insuffisant".

Question difficile ! C'est le niveau Terminale 🎓

La formule dans H2 :

=SI(G2>=16;"Très Bien";SI(G2>=14;"Bien";SI(G2>=12;"Assez Bien";SI(G2>=10;"Passable";"Insuffisant"))))

Comment ça fonctionne ? On imbrique des SI dans des SI :

→ SI moy ≥ 16 → "Très Bien"

→ SINON, SI moy ≥ 14 → "Bien"

→ SINON, SI moy ≥ 12 → "Assez Bien"

→ SINON, SI moy ≥ 10 → "Passable"

→ SINON → "Insuffisant"

On commence toujours par la condition la plus haute (≥16) et on descend. Si on commence par ≥10, tout le monde serait "Passable" !
7
Combien d'élèves ont été admis ? Quelle formule utiliser ?

On utilise la fonction NB.SI() qui compte les cellules respectant un critère :

=NB.SI(H2:H9 ; "Admis")

Elle parcourt la colonne H (Mention) et compte toutes les cases contenant le mot "Admis".

Résultat : 7 élèves admis (NDJOCK Paul seul refusé avec 8.75 de moyenne).
